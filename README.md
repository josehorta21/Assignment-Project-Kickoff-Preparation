# EdgeIQ — Project Kickoff

This repository contains the Project Kickoff documentation for **EdgeIQ**, a sports intelligence and value-betting platform.

It was produced as part of a course assignment on project initiation, following the structure of the [Atlassian Confluence Project Kickoff template](https://www.atlassian.com/software/confluence/templates/project-kickoff) and the principles from Atlassian's [guide to Project Kickoffs](https://www.atlassian.com/work-management/project-management/project-kickoff).

## About the Project

EdgeIQ calculates independent statistical win probabilities for MLB, European football (EPL and La Liga), and NHL games, and compares those probabilities against live market odds to surface statistically positive expected value (+EV) opportunities. The product targets retail sports fans who want a transparent, reproducible model rather than opaque "pick" services.

The MVP is planned as a web platform built on Node.js and Express on the backend, React and Vite on the frontend, PostgreSQL for persistence, and Vercel plus Render for deployment. External data is sourced from API-Sports, The Odds API, and the public NHL API.

## Kickoff Document Structure

The kickoff document covers the key elements expected at project initiation — the who, what, where, when, why, and how — organized into the following sections:

1. Date and Proposed Participants
2. Introductions Context
3. Project Background and Purpose (elevator pitch, SMART goals)
4. Project Definition (in scope, out of scope, deliverables)
5. The Team and Stakeholders
6. Plan and Timeline (milestones, success metrics)
7. Ways of Working (tools, reporting, collaboration)
8. Risk and Issue Management
9. Assumptions
10. Immediate Next Steps
11. Open Questions
